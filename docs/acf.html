Advanced Custom Fields

Relationship field

For relationship fields made in Advanced Custom Fields, use the [related]  shortcode.

Display titles and thumbnails of related posts

[related field="relationship"]     [field title]     [field thumbnail] [/related]

Gallery field

For gallery fields, use the [acf_gallery]  shortcode.

Display images with title

[acf_gallery field="images"]     [acf_image]     [acf_image field="title"]     [acf_image size="thumbnail"] [/acf_gallery] The [acf_image] shortcode displays each image in the gallery field. It can also display these fields: id, title, caption, alt, url, and description. You can set the size parameter to: thumbnail, medium, large. The default is full-size. You can also pass the image IDs to another shortcode to display.

Pass the images to another shortcode

[pass acf_gallery="images"]     [isotope_gallery ids="{FIELD}"] [/pass]

Repeater field

For repeater fields made in Advanced Custom Fields, use the [repeater] shortcode. You can display sub-fields using the [sub] shortcode.

Display repeater fields

[repeater field="boxes"]     [sub field="title"]     [sub image="image"]     [sub field="description" format="true"] [/repeater] For an image field, use the image parameter to display the field. You can set the size parameter to: thumbnail, medium, large - default is full-size. Advanced Custom Fields gives you an option to return the image value as: image ID, image URL, or image object. You can set the in parameter as: id, url, or image object. The default is image object.

Flexible content field

[flex field="flexible_content"]       [layout name="title_text"]         [sub field="title"]         [sub field="text"]     [/layout]       [layout name="title_image_text"]         [sub field="title"]         [sub image="image" size="thumbnail"]         [sub field="text"]     [/layout]       [layout name="gallery"]         [acf_gallery field="gallery" sub="true"]             [sub_image size="thumbnail"]         [/acf_gallery]     [/layout]   [/flex]

ACF fields in other posts

To display gallery, repeater, or flexible content fields from other posts, use the loop shortcode: [loop name="hello-world"]     [repeater field="boxes"]         [sub field="title"]         [sub image="image"]         [sub field="description" format="true"]     [/repeater] [/loop]
